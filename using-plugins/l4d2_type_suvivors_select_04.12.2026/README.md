# Description:

This simple plugin only lets you type in chat to change to map supported survivors.

# Features:
- May fix/ prevent survivor models, vocalize bugs (e.g. When using traditional characters changing mods and playing as Zoey on L4D2 maps may cause the player to experience: bugged model (Zoey's hands but Rochelle's body, voice,... Same go to other survivors as well and vice versa)).
- May fix/ prevent players get stuck/ teleported to L4D1 survivors hold out spots (e.g. When using traditional characters changing mods and if you're playing as Francis when playing The Passing campaign, you might get stuck/ teleported onto the bridge, hold out balcony in the finale chapter).
- No need to install 3rd party character changing related bug fix plugins.
- Detect/ prevent players changing to survivors who are not supported on L4D2 maps and vice versa (Works with custom maps).

# Player Commands:
- !coach as Coach the survivor (or simply type !c, works with other survivors)
- !nick
- !ellis
- !rochelle
- !bill
- !francis
- !louis
- !zoey

# Convars:
- None, as it's always on.
